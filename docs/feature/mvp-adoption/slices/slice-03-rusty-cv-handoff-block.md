# Slice 03 — Add rusty_cv_creator handoff block to evaluation reports

**Goal**: Add `## H) Prepare` block to oferta and auto-pipeline output so reports
contain a ready-to-run rusty_cv_creator command.

## IN scope
- `modes/oferta.md`: add H) Prepare section to report format spec
- `modes/auto-pipeline.md`: add H) Prepare to Step 2 (report save spec) — this replaces
  the now-removed Step 4 (Draft Application Answers)
- Quote generation rule: one sentence ≤120 chars, tailored to JD, in JD's language

## OUT scope
- rusty_cv_creator installation or configuration — user's responsibility
- Batch mode (batch-prompt.md) — separate slice if needed later
- i18n handoff block variants — deferred

## Integration contract (from feature-delta.md)
```
rusty_cv_creator insert \
  --job-title   "<job_title>"     # from JD header
  --company-name "<company>"     # from JD header
  --quote       "<one-line pitch>" # generated by scoring agent
```
Template placeholders: BLANKPOSITION → job_title, BLANKQUOTE → quote

## Learning hypothesis
Disproves: "The quote field is too ambiguous to generate consistently from a JD."
Confirms: a 1-sentence tailored pitch can be derived from any A-G evaluation.

## Acceptance criteria
- AC1: Every report generated by oferta includes `## H) Prepare`
- AC2: H block contains complete `rusty_cv_creator insert` command with all 3 fields populated
- AC3: `--quote` is ≤120 characters, single sentence
- AC3b: H block notes that `--quote` injection is **currently inactive** in rusty_cv_creator
  (`_change_quote_in_destination_cv` call is commented out in file_handlers.rs); block documents
  the fix needed — only `job_title` replaces `BLANKPOSITION` until that call is activated
- AC4: H block carries the label: "Copy-paste to generate your CV. Invocation is manual —
  career-ops does not call rusty_cv_creator automatically. Review the report score before proceeding."
- AC5: `test-all.mjs` checks for `## H) Prepare` presence in any report in `reports/`

## Dependencies
Slice-01 and Slice-02 not technically required first, but logical order maintained.

## Effort estimate
~2 hours (template addition + test assertion)

## Pre-slice SPIKE
None — rusty_cv_creator contract confirmed from src/models.rs + cli_structure.rs reading.
